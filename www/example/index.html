Where is Marigold?

Margold is at example.com and is listening to the same 8080 port as the Webserv server.

Features

  • Static file serving (GET requests)
  • Directory listing (autoindex)
  • File uploads (POST requests)
  • File deletion (DELETE requests)
  • CGI script execution (PHP, Python)
  • HTTP redirections
  • Custom error pages

HTTP Methods

  • GET - Retrieve resources
  • POST - Submit data / upload files
  • DELETE - Remove resources
