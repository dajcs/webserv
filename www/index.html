What is Webserv?

Webserv is a minimal HTTP/1.1 web server built in C++98. It uses non-blocking I/O with a single poll() event loop to handle multiple simultaneous connections efficiently.

Features

  • Static file serving (GET requests)
  • Directory listing (autoindex)
  • File uploads (POST requests)
  • File deletion (DELETE requests)
  • CGI script execution (PHP, Python)
  • HTTP redirections
  • Custom error pages

HTTP Methods

  • GET - Retrieve resources
  • POST - Submit data / upload files
  • DELETE - Remove resources
